## HW_class_Time 
1. Создайте директорию для сборки (находясь в дирректории HW_class_Time):
   ```sh
   mkdir build
   cd build
   ```
2. Запустите CMake для конфигурации проекта :
   
   ```sh
   cmake ..
   ```
   

3. Соберите проект с помощью make:
   ```sh
   make
   ```
4. После успешной сборки проекта вы можете запустить код:
   ```sh
    /class_time
   ```
